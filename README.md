# stevenleabo.com

My personal website. Built with Jekyll and hosted on vercel.

## Getting Started
- Clone or download the repo
- `cd` into the directory
- Run `bundle install`
- Run `bundle exec jekyll serve`

## Post Frontmatter:
```
---
title: "Post title"
description: "Post Subtitle"
date: "YYYY-MM-DD"
tags: ['tag', 'tag2']
featured: true
image: "img.png"
imagethumb: "img-thumb.webp"
layout: post
---
```
