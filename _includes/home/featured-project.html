MagicMirror Modules

MagicMirror is a highly sought-after open-source project that enables users to create a smart mirror utilizing an old monitor and a Raspberry Pi. It is coded in JavaScript and operates on Node.js. This is an excellent opportunity to commence your open-source journey. Feel free to explore the modules I have developed below.


This module displays a random word from the urbandictionary API and refreshes every 3 minutes.

MMM-urbandictionary

This module tells the user if today is Friday the 13th by fetching the date and day in javascript.

MMM-friday-the-13th
