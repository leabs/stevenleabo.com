MagicMirror Modules

MagicMirror is a highly contributed to open-source project that enables users to create a smart mirror utilizing an old monitor and a Raspberry Pi. Mine looks like this. It's coded in JavaScript and operates on Node.js. This is an excellent opportunity to commence your open-source journey. Feel free to explore the modules I have developed below.


MMM-urbandictionary

This module displays a random word from the urbandictionary API and refreshes every 3 minutes.

MMM-friday-the-13th

This module tells the user if today is Friday the 13th by fetching the date and day in javascript.
