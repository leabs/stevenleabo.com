MagicMirror Modules

MagicMirror is a widely supported open-source project that empowers users to transform an old monitor and a Raspberry Pi into a smart mirror. It is programmed using JavaScript and runs on Node.js. This presents an exceptional opportunity to embark on your open-source journey. Please feel free to explore the modules I have developed below.


MMM-urbandictionary

This module displays a random word from the urbandictionary API and refreshes every 3 minutes.

MMM-friday-the-13th

This module tells the user if today is Friday the 13th by fetching the date and day in javascript.
